# **1\. 🎯 Project Brief**

### **1.1 Purpose**

교회의 운영과 관리를 위한 통합 관리 시스템 개발  
행정 업무를 자동화, 간소화  
교회의 성도 관리 강화

### **1.2 Goal**

* 기존의 오프라인 중심 교회 운영을 **디지털화**  
* 목회자/행정팀의 **운영 효율성 향상**  
* 성도의 **참여도 및 소속감 증대**  
* 추후 **모바일 앱으로 확장**하여 교회 내 소셜 네트워크화

## **5\. Work Flow**

## **5.1 관리자(목사님/개발자) 사용자 흐름 (초기 1회)**

* ### **초기 세팅 플로우** 

  * 로그인  
  * 교회 선택  
    * 시티 주안교회  
  * 조직 구조 등록/확인  
  * 목장 생성  
    * 대학목장  
  * 셀 생성  
    * (00셀, 01셀…)  
* 리더(셀장/부셀장) 계정 초대 또는 권한 부여  
* 기본 성도 Type/등급 옵션 확인  
  * 새신자 타입 A/B/C  
  * 성도 타입 D/E/F/G  
* **결과:** 운영 가능한 기본 환경 완성

### **5.2. 새신자 등록 플로우 (가장 자주)**

* ## 로그인 → “새신자 등록” 진입

* ## 새신자 정보 입력

  * ## 이름, 연락처

  * ## 새신자 타입(A/B/C)

* ## 기본값 자동 적용

  * ## 상태: 새신자

  * ## 소속: 미배정

* ## 저장

* ## 등록 완료 화면에서 바로 다음 액션 선택

  * ## “셀 배정하기”

  * ## “평가 입력하기”

  * ## “목록으로 돌아가기”

* ## **결과:** 새신자 데이터가 DB에 누적됨

### **5.3. 셀/목장 배정 플로우 (새신자반/셀 분배)**

* ## “새신자 목록” → 미배정 필터 선택

* ## 새신자 선택

* ## “소속 변경” 클릭

* ## 교회/목장/셀 선택

* ## 저장

* ## 변경 이력 자동 기록

  * ## 언제, 누구에 의해, 어디로 이동했는지

* ## **결과:** 새신자가 조직 구조에 편입 \+ 히스토리 축적

### **5.4. 조회/검색/정렬 플로우 (관리 리포트)**

* 대시보드/목록 화면 진입  
* 검색  
  * 이름 검색  
  * 그룹(목장/셀) 필터  
* 정렬(sorting)  
  * 목장별  
  * 셀별  
* 범위 선택  
  * 셀 전체 보기  
  * 목장 전체 보기  
  * 교회 전체 보기  
* **결과:** “2개 셀 운영”이 가능한 관리 화면 확보

## **5.5 셀장/부셀장(리더) 사용자 흐름**

* ### **내 셀 멤버 보기 플로우 (매주/수시)**

  * 로그인  
  * “내 셀” 자동 진입 (권한 범위 제한)  
  * 셀원 목록 확인  
  * 멤버 클릭 → 상세 정보 열람  
    * 기본 정보  
    * 현재 소속  
    * 최신 평가/등급  
    * 이동 히스토리(읽기 전용)  
* **결과:** 리더는 자기 셀 운영에 필요한 정보만

## **5.6 MVP 시연용 ‘대표 시나리오’ (2개 셀 파일럿)**

* **DEMO 시나리오**  
  * 관리자 로그인 → 조직 구조(대학목장/2개 셀) 확인  
  * 새신자 1명 등록(A타입)  
  * 새신자 셀 배정(00셀)  
  * 셀장 로그인 → 내 셀에서 새신자 확인  
  * 관리자 화면에서  
    * 해당 셀 전체 보기  
    * 등급별 필터/정렬  
    * 새신자→일반 성도 전환까지 시연

## **5.7 플로우를 고정하면 자동으로 결정되는 “필수 화면 6개”**

* **6개의 MVP**  
  * 로그인  
  * 내 셀(리더 홈) / 관리자 대시보드(관리자 홈)  
  * 새신자 등록 폼  
  * 멤버 목록(검색/필터/정렬)  
  * 멤버 상세(평가/상태/소속/히스토리)  
  * 조직 관리(목장/셀 생성 \+ 리더 권한)
